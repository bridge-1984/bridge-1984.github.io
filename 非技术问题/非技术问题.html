    # 1. 自我介绍

    面试官好，我叫xx，来自xx大学，就读的专业是软件工程。
    在学校中，我学习过HTML、CSS、java程序设计、计算机网络、数据结构等课程，并根据自己的兴趣选择了前端进行更深入的学习。 在前端学习的过程中，我也做了一些项目以巩固自己的知识，其中有两个项目我有比较多的收获。 一个是我和其他6位同学协作完成的火车购票网站，在该项目中，我负责了前端的大部分工作，包括页面的设计和前后端数据交互的实现，同时积极地与项目成员进行沟通以保证项目进展顺利。（该项目让我对小组协同、前后端分离开发的方式有了更深的了解。）而另一个则是个人完成的博客网站。(该网站使用vue+elementUI+node+mysql实现，分为文章展示和文章管理两大部分。)开发这个网站一是为了能够更好地记录自己学习的内容，二是为了检验自己对于vue框架的学习成果。通过这个网站的开发，我巩固了vue的知识，也弥补了一些欠缺的知识，比如vuex的使用，同时也锻炼了自己解决问题的能力。 在学校我也参加一些了学生活动，比如大一时我加入了学院团委的宣传部，大二开始担任班里的宣传委员到现在 在性格上，我认为自己是一个有责任心且善于倾听的人。对于自己负责的任务，我会认真及时地完成，对于别人的建议，我能用心思考并虚心接纳，在与其他人共同完成项目的过程中，我也能和他们保持良好的沟通。 以上就是我的自我介绍，谢谢

    # 2. 为什么选择前端

    我在自我介绍里也有提到，一开始学前端是因为兴趣，第一次用html写出一个自己的静态页面让我觉得很有意思，所以我选择了前端进行进一步的学习。而在深入学习前端后，我发现前端并不是像有些人说的那样没什么技术含量，并且前端的技术更新很快，不像后端技术相对固定了，我很想知道前端会发展成什么情况，所以选择了前端。

    # 3. 平时怎么学习前端的，有看什么书吗

    学校的课程中涉及到前端的内容比较少，所以我主要是通过自学。
    平时主要是通过看视频和廖雪峰、阮一峰等技术大牛的博客，并结合官方文档进行学习，也会看一些书，比如js高程、正则表达式迷你书、图解http等来巩固自己的知识。

    # 4. 对我们公司有什么了解吗

    兴业数金是兴业银行集团旗下金融科技子公司，全面负责兴业银行集团科技研发和数字化创新工作。公司致力于运用云计算、区块链、人工智能、开放API、流程机器人等前沿科技，为商业银行数字化转型提供解决方案、输出科技产品与服务，在上海、福州、广州、深圳等多个城市都设有办公地点

    # 5. 对金融行业的了解，和互联网公司的区别

    我觉得最大的区别是互联网公司比较看重技术的广度和深度，在实现上会更愿意使用新的技术进行尝试；而金融行业更看重业务的实现，在技术选择方面相对固定

    # 6. 大概介绍一下你做项目的一个步骤

    1）需求分析，确定整个项目需要实现的功能模块
    2）针对需求进行技术选型，选择合适的技术或框架进行开发
    3）开发完项目后，再对每个功能进行测试，保证功能能正确实现
    4）有需要的情况下对项目代码进行优化
